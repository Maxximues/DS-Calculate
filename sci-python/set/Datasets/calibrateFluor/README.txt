File calibrateFluor.mat: Variable y0 gives total fluorescence of many cells prior to mitosis (arbitrary units). Variable Deltay gives difference in fluorescence between that cell's two daughters, in the same arbitrary units.

Data courtesy Michael Elowitz; see also Fig 2B of Rosenfeld et al. Gene regulation at the single-cell level. Science (2005) vol. 307 (5717) pp. 1962-1965.
