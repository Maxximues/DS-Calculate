The variable Nsee(i,j,k) gives the number of times that stimulus i (1 <= i <= 3) elicited rating j-1 (0 <= j-1 <= 6)  from subject k (=1,2,3). 
The variable photonsin(i) gives the strength of stimulus #i in mean number of photons presented to the cornea.
